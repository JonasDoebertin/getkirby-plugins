Title: Auto Publish

----

Subtitle: Make new Pages Visible by Default

----

Description: Extends Kirby 2.1 with the possibility to auto publish pages that are created with the Kirby Admin Panel. By default new pages are invisible on creation. This plugin allows you to make all new pages or certain types of pages visible by default.
